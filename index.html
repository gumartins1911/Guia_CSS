Regra da Cascata

As regras da cascata funciona como se fosse uma hierarquia dentro do CSS, fazendo com que a ultima declaração seje sempre a executada.

A cascata ajuda a resolve conflitos dentro do no CSS considerando:

  1. Origem do estilo
  2. Especificidade
  3. Ordem de Declaração
  4. Herança

Especificidade

A Regra de Especificidade dentro do CSS uma regra que o navegador utiliza para que possa decidir qual o estilo aplicar quando múltiplas regras diferentes tentam afetar o mesmo elemento.

  • Tipo de Seletor:
  1. Seletor de elemento
  2. Seletor de classe, atributo, pseudo-classe
  3. Seletor de ID
  4. Estilo inline (no HTML)
  1. 0,0,1
  2. 0,1,0
  3. 1,0,0
  4. 1,0,0,0

Combinadores

Combinadores são símbolos que são utilizados no CSS para definir a relação entre dois seletores HTML. Este elemento nos permite selecionar elementos com base em sua posição ou hierarquia dentro do código HTML.

  • Evita repetir classes oi IDs
  • Aumenta a clareza e organização do CSS
  • Permite aplicar estilos específicos sem mexer no HTML
  • Deixa o CSS mais limpo e poderoso

pseudo-classe

Uma pseudo-classe é basicamente, uma palavras que é adicionada no fim de um seletor CSS, antecipada pos dois pontos (:). Isso nos permite aplicar estilos a elementos HTMLque estão em um estada específico, sem peecisar adicionar clases nem alterar HTML.

Lista de pseudo-classes úteis:

First-Child:

é usada para selecionar o primeiro filho de um elemento pai. Isso significa que o estilo será aplicado somente ao primeiro elemento dentro de um contêiner, se ele for do tipo que você especificou.

last-child

serve para selecionar o último filho de um elemento pai. Ou seja, o estilo será aplicado somente ao último elemento dentro de um contêiner, desde que ele seja do tipo correto que você estiver selecionando.

nth-child

serve para selecionar um ou mais filhos com base na sua posição (ordem) dentro de um elemento pai.

nth-of-type

é usada para selecionar o enésimo elemento de um determinado tipo de tag dentro de um elemento pai.

inline-clock

Inline-blocké um valor da propriedade CSS display que combina características de elementos inline e block.

Onde utilizar?

  • Para criar menus horizontais
  • Para alinhar caixas ou cards lado a lado
  • Para estilizar botões ou links com tamanho ajustável
  • Quando precisa do comportamento visual de inline, mas controle como block

Margin, Padding e Border

  • Padding: Espaço interno. Espaço entre a borda e o conteúdo.
  • Border: Elemento que contorna todo o conteúdo.
  • Margin: Espaço fora do elemento ou espaço entre o elemento.
